# LiveReload — Portable Automatic Browser Refresh on Code Change

## About LiveReload

LiveReload provides a complete live reloading environment for front‑end developers who want instant feedback. LiveReload combines file watching, browser injection, and automatic refresh. LiveReload enables users to save HTML, CSS, JavaScript, and see changes immediately without manual refresh. LiveReload runs on desktop platforms. LiveReload is trusted by web developers, front‑end engineers, CSS designers, and productivity enthusiasts worldwide.

## Deep Dive — Live Reloading Notes

File watching: monitor HTML, CSS, JS, PHP, images, any file extension
Browser injection: inject CSS/JS without full page reload (CSS changes)
Cross‑browser: Chrome, Safari, Firefox, Edge, Opera, mobile devices
Customization: delay before reload, exclude files, custom commands
Integration: Sublime Text, VS Code, Brackets, any editor with save hook
